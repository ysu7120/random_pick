# 🍀 행운의 뽑기판 (Lucky Pick Board)

10x10 크기의 총 100개 뽑기 카드로 구성된 웹 기반 인터랙티브 뽑기판 프로그램입니다. 1등부터 5등까지의 등수별 상품을 자유롭게 입력하여 이벤트를 진행할 수 있으며, 직관적인 UI와 실시간 상태 보존 기능, 화려한 선물 개봉 연출을 제공합니다.

---

## 🚀 주요 기능

### 1. 10x10 뽑기 카드 그리드
- 총 100개의 네잎클로버(🍀) 카드로 구성된 직관적인 인터페이스를 제공합니다.
- 카드를 클릭하면 개봉된 등수가 노출되며, 한 번 뽑은 카드는 다시 클릭할 수 없도록 방지됩니다.

### 2. 등수별 상품 설정 및 실시간 개수 집계
- **등수별 수량 고정**: 1등(2개), 2등(10개), 3등(15개), 4등(30개), 5등(43개)으로 총 100개의 당첨 카드로 구성되어 있으며, 꽝이 존재하지 않습니다.
- **상품명 커스텀 입력**: 각 등수 좌측의 입력란에 상품명을 자유롭게 기록할 수 있으며, 입력 사항은 실시간 저장됩니다.
- **실시간 남은 수량 표시**: 우측 사이드바 영역에서 등수별로 남은 수량이 실시간으로 집계 및 표시됩니다.

### 3. 몰입감 넘치는 선물 상자 개봉 연출 (결과 모달)
- **흔들림 애니메이션**: 카드를 클릭하면 선물 상자가 흔들리는 티저 애니메이션이 수행됩니다.
- **특별 등수 연출 (1, 2등)**: 
  - 1등과 2등 당첨 시에는 선물 상자가 더욱 역동적이고 길게 흔들립니다.
  - 특히 **1등**의 경우, 개봉 전 상자가 붉은빛으로 강하게 진동하는 특별 연출이 가미되어 기대감을 고조시킵니다.
- **종이 폭죽 (Confetti) 이펙트**: 선물 상자가 열리면서 알록달록한 종이 조각들이 터져 나오는 애니메이션 효과가 표시됩니다.
- **결과 표시**: 상자가 사라진 후, 축하 메시지와 함께 설정해 둔 등수별 상품 정보가 모달에 상세하게 노출됩니다.

### 4. 로컬 스토리지(LocalStorage) 상태 영구 보존
- 카드의 셔플 상태(deck), 각 카드의 개봉 상태(opened), 입력된 등수별 상품 정보가 웹 브라우저의 로컬 스토리지에 즉시 자동 저장됩니다.
- 브라우저를 새로고침하거나 탭을 닫고 다시 접속해도 기존에 진행하던 뽑기 내역이 완벽히 복원됩니다.

### 5. 뽑기판 초기화 (다시 뽑기)
- '다시 뽑기' 버튼을 누르면 입력한 상품 정보는 그대로 유지한 채로, 카드의 위치를 무작위로 다시 섞어(Fisher-Yates Shuffle) 새 게임을 즉시 시작할 수 있습니다.

## 📂 프로젝트 구조

```text
c:\workspace\random_pick\
├── index.html        # 마크업 구성 (공원 배경 데코레이션, 입력 폼, 그리드 영역, 결과 모달)
├── style.css         # 테마 디자인 스타일링, 반응형 그리드 레이아웃, 애니메이션(Confetti, Shake, Bloom) 키프레임
└── script.js         # 상태 제어 logic (Fisher-Yates 셔플, 로컬스토리지 저장/복원, 타이머 기반 모달 연출 제어)
```

### 주요 소스 코드 설명

- **[index.html](file:///c:/workspace/random_pick/index.html)**
  - 공원 분위기를 연출하는 클로버, 벌, 나비, 꽃 등의 이모지 데코레이터 배치.
  - 상품 입력폼, 뽑기판 영역, 결과창 모달이 접근성을 고려하여 HTML5 시맨틱 태그 및 `aria-*` 속성으로 구현되었습니다.
- **[style.css](file:///c:/workspace/random_pick/style.css)**
  - 미개봉 카드 hover/focus 효과, 개봉 시 플립 느낌의 색상 전환 스타일링.
  - 모달 뒤쪽 배경을 블러 처리하는 CSS Backdrop Filter 적용.
  - 1등 당첨 시 긴장감을 주는 `@keyframes shake-strong` 및 붉은 후광 효과 구현.
- **[script.js](file:///c:/workspace/random_pick/script.js)**
  - Fisher-Yates 알고리즘을 사용한 카드 위치 무작위 셔플링 (`shuffle()` 함수).
  - 모달 노출 중 중복 클릭을 차단하기 위한 상태 플래그 제어.
  - 브라우저 쿠키나 세션 만료와 관계없이 영구적인 기록을 보존하는 `saveState()` 및 `loadState()`.

3. 1등~5등 상품명을 채워 넣은 뒤 신나게 카드를 골라보세요!
